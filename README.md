## how to play    
1. ### player 1    
- W = jump    
- A = left    
- S = duck   
- D = right   
- Z = attack    
- T = blocking    
- R = throw weapon
    
    
2. ### player 2    
- Up Arrow = jump    
- Left Arrow = left    
- Down Arrow = duck    
- Right Arrow = right    
- O = attack    
- P = blocking    
- I = throw weapon
